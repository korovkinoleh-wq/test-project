# CV Summary

Ansvarlig og initiativrik kandidat med erfaring fra arbeid i Norge, praktisk bakgrunn innen isolasjon og teknisk interesse for digitale verktøy, AI og automatisering. Har førerkort klasse B, egen bil og erfaring med kjøring i Norge. Bekreftet utdanning gjennom NAKUT innen automatiserte styringssystemer. Lærer raskt, jobber strukturert og trives med oppgaver innen practical work, logistics, support, automation og enkle dashboard-/workflow-relaterte løsninger.
